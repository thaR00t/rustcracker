# RustCracker
Hash cracker but written in rust :)

````
                                          █▀█ █ █ █▀▀ ▀█▀ █▀▀ █▀█ ▄▀█ █▀▀ █▄▀ █▀▀ █▀█
                                          █▀▄ █▄█ ▄▄█  █  █▄▄ █▀▄ █▀█ █▄▄ █ █ ██▄ █▀▄
````


Building the packets:

````
cargo build --release
````
If you'd like to do it automatically you can run:
````
sudo ./install.sh
````

Usage:
````
rustcracker [OPTIONS] --hash <HASH> --wordlist <WORDLIST>
````
